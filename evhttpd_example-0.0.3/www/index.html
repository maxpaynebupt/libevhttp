Auto-scaling prototype

NFV vIMS PoC: Auto-scaling Proof-of-Concept

This demonstration has been tested with Mozilla Firefox and Google Chrome. It does not work with Internet Explorer.

Please adjust the zoom level of the "view pages" so that the text is as readable as possible on the screen that you are using for the demonstration. In Firefox, you can adjust the zoom level with "Ctrl +" (plus) and "Ctrl -" (minus) and you can reset to the default zoom level with "Ctrl 0" (zero).

Remember to clear the console from time to time so that the number of messages does not become too large. This will also make scrolling easier.
